# Vim Config

## Plugin Manager

install `vim-plug`

```
curl -fLo ~/.vim/autoload/plug.vim --create-dirs \
https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim
```

link js, jsx ftplugin

```
ln -s ./ftplugin/javascriptreact.vim ./ftplugin/javascript.vim
```
